# How to install ? 

<pre>
1. clone this repo
2. sudo bash install.sh
</pre>

# How to run ?

<pre>
1. install this app into your phone (https://github.com/FajarTheGGman/Raspi-Hungry/raw/master/Raspi-Hungry.apk)
2. Just Type "npm start" in raspberry pi 
</pre>
